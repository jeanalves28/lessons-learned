Sobre o meu aprendizado.

Durante essas três semanas de curso, houveram muitos aprendizados e "mão na massa"! No primeiro bloco, pude aprender um pouco mais sobre os Fundamentos de Desenvolvimento Web, aprendendo sobre o sistema operacional Unix e a ferramenta de script Bash. Além disso, já no segundo bloco, entendemos o que é e para o que serve o Git, Github e Internet, fazendo, inclusive, a criação dos perfis no Github. No bloco 3, começamos com uma introdução ao HTML, que é a estrutura (esqueleto) de uma página web e ao CSS, que vai nos garantir a aparência da página.
