Meu aprendizado

O HTML é considerado o esqueleto de uma página web. Aprendi que, através desta linguagem de marcação de texto, posso criar estruturas de como a página será exibida utilizando as tags, estruturando textos e títulos, criando listas, links e adicionando imagens.

O HTML e o CSS são duas linguagens complementares, que me permitem criar páginas web do zero. Foi através do CSS que consegui estilizar a minha página.

Além destas linguagens, aprendi que devo desenvolver páginas web onde qualquer pessoa possa perceber, entender, navegar e interagir.
